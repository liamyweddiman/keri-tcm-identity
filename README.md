# KERI-TCM-Identity  
### A KERI-based Identity Framework for TCM Practitioners & Traditional Medicine Provenance  
**Author: Liam YW Eddiman**

## ğŸ” Overview
This repository explores how **KERI (Key Event Receipt Infrastructure)** can support a trustworthy, verifiable identity infrastructure for the Traditional Chinese Medicine (TCM) ecosystem.

The project focuses on two core domains:

1. **TCM Practitioner Credential Governance**
   - Apprenticeship & lineage attestation  
   - Institutional training & certification records  
   - Cross-border medical license verification  
   - Decentralized clinical activity & renewal logs  

2. **Traditional Medicine & Herbal Product Provenance**
   - Farm-level origin authentication  
   - Processing & extraction event chains  
   - Laboratory testing & batch attestation  
   - Export certification & pharmaceutical integration
